# Foundry 🔍

> **Lost something? Found something? Let's reunite it.**

**Foundry** is a modern, student-centric web application built to serve as a digital Campus Lost & Found Board. It provides a simple, interactive platform for students and staff to post lost or found items, track claims, and reunite belongings with their rightful owners.

---

## ✨ Features

- 📌 **Post Lost & Found Items:** Easily create posts with details like item category, location, and description.
- 🔍 **Search & Quick Filters:** Search items by keywords or filter by common categories (*Electronics, Keys, Wallet, Bag, ID Card*).
- 🏷️ **Status Tracking & Board Views:** Browse posts organized by status—**All**, **Lost**, **Found**, and **Claimed**—with sorting options (*Most Recent*, *Oldest First*, *Title A-Z*).
- 📊 **Real-time Statistics:** Get a quick overview of campus metrics (Total Items, Lost, Found, Reunited).
- ⚡ **Recent Activity Feed:** Stay updated with a live timeline of recent posts and claimed items.
- 🔥 **Trending Section:** Highlight popular items receiving user interaction (comments and saves).
- 💬 **Interactive Comments:** Coordinate item pickups directly through item-level discussions.
- 🌓 **Dark Mode Support:** Built-in theme toggle for seamless viewing day or night.
- 📈 **Visitor Counter:** Track total engagement and page visits.
